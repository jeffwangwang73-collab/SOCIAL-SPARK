# Contributing to Give Me a Light (借个火)

We love your input! We want to make contributing to this project as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features
- Becoming a maintainer

## Development Process

1. **Setup:** Ensure you have your `API_KEY` for Gemini services.
2. **Coding:** Follow the established UI/UX patterns (warm tones, Jakarta Sans font).
3. **Commit Messages:** Use descriptive commit messages (e.g., `fix: map marker jumpiness`, `feat: added reputation badges`).
4. **Pull Requests:** If you're working in a team, submit a PR and wait for a review.

## Community Guidelines

This is a community for enthusiasts. Please ensure all contributions:
- Respect local laws and tobacco regulations.
- Promote a helpful and friendly environment.
- Prioritize user safety and privacy.

Thank you for helping us point the way to a warmer community!
