# 借个火 (Give Me a Light) - Social Spark

**借个火 (Social Spark)** 是一款温暖的、社区驱动的烟友社交应用。它通过现代、美观的界面，促进基于位置的互动、社区分享和互助。

---

## 🚀 核心功能 (Core Features)

- **寻火 (Location Discovery):** 实时地图界面，发现附近的烟友和吸烟点。
- **互助系统 (Mutual Help):** 向附近成员借火、借烟或仅仅是发起聊天。
- **信誉系统 (Reputation):** 通过帮助他人获得“黄金”等级和积分，构建互信社区。
- **AI 助手 (AI Features):** 集成 Gemini API，提供二手价格建议和动态内容质量评估。
- **社区中心 (Community Hub):** 探索品牌故事、器材指南和同城聚会热点。
- **安全保障:** 18+ 年龄门槛 (Age Gate) 与社区守则。

## 🛠️ 技术栈 (Tech Stack)

- **前端:** React 19, Tailwind CSS
- **路由:** React Router 7
- **图标:** Google Material Symbols
- **人工智能:** Google Gemini API (@google/genai)
- **视觉风格:** 自定义 "Glassmorphism" (玻璃拟态) 与 暖色调设计系统

## 📂 项目结构

- `pages/`: 包含所有功能页面（登录、首页、地图、聊天等）。
- `services/`: AI 相关逻辑。
- `types.ts`: 全局类型定义。
