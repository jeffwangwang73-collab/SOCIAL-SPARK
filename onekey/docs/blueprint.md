# **App Name**: 安全认证助手

## Core Features:

- 动画加载指示器: 在认证过程中显示一个环绕盾牌图标旋转的动画，指示系统正在处理中。
- 一键认证按钮: 提供一个颜色鲜明的“一键认证”按钮，用户点击后触发认证流程。
- 服务条款和隐私政策链接: 用户在认证前需要勾选同意服务条款和隐私政策，提供相应的链接文本。

## Style Guidelines:

- 主色：鲜艳的橙色 (#FF7800)，象征活力和信任。
- 背景色：浅灰色 (#FAFAFA)，为内容提供清晰的视觉基础。
- 强调色：深橙色 (#E64A19)，用于重点突出和用户互动元素。
- 正文字体和标题字体：使用 'PT Sans'，一种兼具现代感和亲和力的人文主义非衬线字体，适用于标题和正文。
- 使用一个盾牌图标与加号，表示安全认证。
- 页面主要元素垂直居中，确保在各种屏幕尺寸上均能良好显示。
- 围绕盾牌图标的旋转动画，表明认证正在进行中。
