# **App Name**: 验证成功

## Core Features:

- 显示成功图标: 页面中央显示一个绿色圆圈和对勾，代表验证成功。
- 显示验证成功文本: 在图标下方，显示黑色的“验证成功”文本。
- 再体验一次按钮: 在页面底部，显示一个橙色的“再次体验”按钮。
- 重新加载: 再次点击跳转回到重新验证

## Style Guidelines:

- 背景颜色：白色 (#FFFFFF)，创建一个干净、整洁的页面。
- 成功图标颜色：绿色 (#28A745)，表示成功完成操作。
- 按钮颜色：橙色 (#FFBB21)，吸引用户点击。
- 主要字体：'Inter'，用于所有文本，包括标题和按钮文字，确保清晰易读。Inter is a sans-serif font.
- 图标和文本居中对齐，按钮位于页面底部。
- 淡入动画，图标和文本在页面加载时以平滑的动画效果出现。
