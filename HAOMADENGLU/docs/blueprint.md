# **App Name**: 号码验证

## Core Features:

- 手机号码输入: 提供一个输入框，用户可以在其中输入他们的手机号码以进行身份验证。
- 一键认证: 启用使用单一身份验证解决方案(如 SMS OTP)的无密码登录，简化用户流程。
- 阅读并同意: 将复选框和用户协议与隐私政策超链接，表明用户已阅读并接受条款。

## Style Guidelines:

- Primary color: Orange (#FF7F00) for the primary button, reflecting a bright, accessible design. It should strongly contrast with the background.
- Background color: Very light gray (#FAFAFA) for the background to ensure a clean and focused interface.
- Accent color: A lighter shade of orange (#FF9F40) for hover states and other interactive elements, complementing the primary color.
- Font: 'Inter', a grotesque-style sans-serif with a neutral look suitable for body text and headlines. The font provides high readability which will benefit users when reading the '用户协议' and '隐私政策'.
- Center-align all the elements, keeping consistent paddings and margins
- Implement gentle transition effects during button hovers
