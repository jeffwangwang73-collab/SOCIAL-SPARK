# **App Name**: 蜂窝登录

## Core Features:

- 验证码登录: 用户通过输入手机号码和验证码进行登录。
- 密码登录: 用户通过输入密码进行登录
- 手机号码输入: 用户输入他们的手机号码以接收验证码。
- 获取验证码: 用户点击该按钮后，系统会向输入的手机号发送一条验证码。
- 登录: 当用户点击该按钮后，输入的手机号码和验证码会被发送到后端验证。 如果验证成功，用户将可以登录系统。
- 用户协议与隐私政策: 用户必须同意这些文件才能登录

## Style Guidelines:

- 主色: 鲜亮橙色 (#FF7800) 用于按钮和重点强调，突出交互元素。
- 背景色: 浅灰色 (#F5F5F5)，保持页面清洁和专注。
- 正文和标题字体: 'Inter'，一种现代的无衬线字体，使内容易于阅读。
- 页面应有一个居中的登录框，包含手机号输入、验证码输入以及登录按钮。
